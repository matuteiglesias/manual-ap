## Propósito

Definir reglas de limpieza para convertir listados crudos en una base utilizable por operación política.

## Criterios

- Formato único para teléfonos.
- Normalización de nombres.
- Detección y eliminación de duplicados.
- Registro de fuente y fecha de actualización.
