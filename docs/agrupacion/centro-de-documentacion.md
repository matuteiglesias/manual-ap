## Propósito

Este sitio reúne criterios, memoria y herramientas para sostener la práctica política y organizativa de la agrupación.

## Cómo está organizado

- **Manual operativo**: acuerdos de trabajo, carriles, roles y gobernanza.
- **Memorias del chat**: reconstrucciones por período para estudiar decisiones y ritmos.
- **Lecciones**: aprendizajes transversales para corregir coordinación y ejecución.
- **Referencia organizativa**: actores, perfiles y guías de consulta.
- **Anexos metodológicos**: procedimientos y scripts de soporte.

## Regla de lectura

Cada página debe servir para una acción concreta: planificar, coordinar, evaluar o corregir prácticas del grupo.
