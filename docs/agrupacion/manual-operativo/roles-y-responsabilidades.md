## Propósito

Precisar quién hace qué, con qué entregable y con qué frecuencia.

## Principios

- Un rol, una responsabilidad principal.
- Cada responsabilidad tiene respaldo.
- Los handoffs entre roles se registran.

## Señales de buena asignación

- Menos tareas huérfanas.
- Menos dependencia de una sola persona.
- Mayor previsibilidad en acciones públicas.
