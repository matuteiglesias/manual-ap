## Propósito

Alinear roles para reducir superposición, cuellos de botella y tareas sin cierre.

## Roles mínimos

- Coordinación general.
- Coordinación de movilización.
- Coordinación de comunicación.
- Responsable institucional.
- Responsable de memoria y seguimiento.

## Procesos mínimos

- Planeamiento semanal con prioridades acotadas.
- Confirmación logística 48/24/2h antes de cada acción.
- Cierre post-acción con resultados, incidentes y ajustes.
